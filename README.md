Built my first Python project — a modern desktop-based Task Manager application designed to help users organize and manage daily tasks efficiently.

This application was developed using Python with a graphical user interface and local file-based data storage. It allows users to create, view, and delete tasks through a clean and user-friendly interface.

Features Implemented:

> Add and manage daily tasks
> Delete completed or unwanted tasks
> Persistent task storage using file handling
> Data loading when the application starts
> User-friendly desktop interface
> Modern UI design inspired by productivity applications

Technologies Used:
Python 
Tkinter / CustomTkinter (GUI Development)
File Handling
Object-Oriented Programming concepts

This project helped me strengthen my understanding of Python fundamentals, GUI development, event handling, and building real-world desktop applications.

Excited to continue learning and building more advanced software projects.
